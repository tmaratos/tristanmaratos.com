Add-on detail

Content Management System (CMS)

A CMS allows approved users to manage specific website content without editing code.

May include

  • Editable text and content
  • Staff page management
  • Announcements
  • Image updates
  • Menu or service updates
  • Limited user roles
  • Protected login access

Add-on detail

Administrative Dashboard / System

An administrative system is a more advanced operational layer for organizations that need internal tools beyond website content editing.

May include

  • User and employee management
  • Role based access control
  • Scheduling systems
  • Internal workflow tools
  • Form and submission management
  • Analytics and reporting
  • Document or media management
  • Secure authentication
  • Protected administrative routes
  • Database driven infrastructure

Permission tiers

Role examples

Typical permission tiers used inside an administrative system. Final structure depends on the organization and how internal access needs to be separated.

  • Super Admin Full system control
  • Administrator Manage users, content, and settings
  • Manager Operational oversight and limited management access
  • Staff / User Limited workflow access
  • Viewer Read only access
